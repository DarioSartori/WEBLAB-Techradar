## Tech-Stack

**Frontend:** Angular  
**Backend:** NestJS 
**Datenbank:** PostgreSQL + Prisma
**Auth:** JWT  
**Testing:** Cypress + Jest

## Struktur
- frontend/ – Angular App
- backend/  – NestJS API + Tests (Jest)
- cypress/  – Cypress E2E-Tests
